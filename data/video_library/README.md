# Video Library

Drop your reusable background videos in this folder and register them in
`catalog.json`. Each entry needs a unique `code`, a `title`, and the video
`filename` stored in this folder.

Example:

```json
[
  {
    "code": "motivation_01",
    "title": "Motivation Loop 01",
    "filename": "motivation_01.mp4"
  }
]
```
